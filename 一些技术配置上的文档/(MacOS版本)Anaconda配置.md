
# Jupyter notebook 打开方式
## 方式一：Terminal 打开
~~~bash
conda activate base  # 打开bash环境
~~~
~~~bash
jupyter notebook   # 打开jupyter notebook
~~~
### 💡 小技巧：指定 Jupyter 的工作目录

- **方法一（推荐）**：先在终端里 `cd` 到你想作为根文件夹的目录，再执行 `jupyter notebook`。
    ~~~bash
    
    cd /Users/你的用户名/我的数据科学项目
    jupyter notebook
    
    Jupyter 就会以这个文件夹作为根目录。
	~~~
