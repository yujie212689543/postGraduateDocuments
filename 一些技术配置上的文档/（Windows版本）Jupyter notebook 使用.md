# 常用快捷键
#### 命令模式快捷键
- **A**：在当前单元格上方插入新单元格
- **B**：在当前单元格下方插入新单元格。
- **D, D**：删除当前单元格（连续按两次 D 键）。
- **Z**：撤销最近一次删除的单元格。
- **Y**：将当前单元格切换为代码单元格。
- **M**：将当前单元格切换为 Markdown 单元格。
- **Shift + Enter**：运行当前单元格的代码并跳转到下一个单元格。
- **Ctrl + S**：保存当前 Notebook 文件。

#### 编辑模式快捷键
- **Tab**：自动补全代码或方法名。
- **Ctrl + Z**：撤销最近的编辑。
- **Ctrl + Y**：重做最近的编辑。
- **Home**：跳转到当前行的行首。
- **End**：跳转到当前行的行尾。
