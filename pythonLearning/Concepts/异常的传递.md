## 一、`division by zero` 是什么意思？

- `division by zero` 是 Python 中 ** `ZeroDivisionError` ** 异常的默认错误信息。
    
- 当程序尝试执行一个除以零的运算（比如 `1/0`）时，Python 就会抛出这个异常，并附带这条消息。
    
- 在异常处理中，如果你用 `except Exception as e:` 捕获它，变量 `e` 就保存了这个异常对象，打印 `e` 就会输出 `division by zero`。
    

## 二、你的代码执行流程

```python

def func 01():
    print("这是 func 01 开始")
    num = 1/0          # 第 3 行：这里会引发 ZeroDivisionError
    print("这是 func 01 结束")
def func 02():
    print("这是 func 02 开始")   # 注意这里你写的是 FUNC 02 开始，但输出显示 "这是 func 02 开始"
    func 01()                  # 调用 func 01
    print("这是 func 02 结束")   # 不会执行，因为 func 01 异常了
def main():
    try:
        func 02()              # 调用 func 02
    except Exception as e:    # 捕获任意异常
        print(e)              # 打印异常信息
    main()                    # ⚠️ 这里递归调用 main() 自身
```

当你运行 `main()` 时：

1. `main` 调用 `func02`。
2. `func02` 打印 `"这是func02开始"`。
3. `func02` 调用 `func01`。
4. `func01` 打印 `"这是func01开始"`。
5. `func01` 执行 `1/0` → 抛出 `ZeroDivisionError`。
6. 异常向上冒泡：`func01` → `func02` → `main` 中的 `try`。
7. `except Exception as e:` 捕获异常，`e` 就是异常对象，`print(e)` 输出 `division by zero`。
    

所以你的输出是：

```text

这是 func 02 开始
这是 func 01 开始
division by zero
```
