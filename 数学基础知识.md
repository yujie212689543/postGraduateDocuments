# 方差与标准差
## 方差
***定义***:$$\sigma^2 = \frac{1}{N}\sum_{i=1}^N (x_i - \mu)^2$$

## 标准差
- [标准差](https://link.zhihu.com/?target=https%3A//wuu.wikipedia.org/wiki/%25E6%25A0%2587%25E5%2587%2586%25E5%25B7%25AE)**：衡量 一组数据相对于其均值的离散程度**的指标，通常用σ表示。（白话：一堆数离这堆数的平均值的分散程度）
- ***定义***：$$\sigma^2 = \frac{1}{N}\sum_{i=1}^N (x_i - \mu)^2$$
